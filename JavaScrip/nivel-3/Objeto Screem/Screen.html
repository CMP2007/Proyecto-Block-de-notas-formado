Objeto Screen

Se trata de un objeto nativo de JavaScrip, al cual permite acceder a ciertas propiedades de la pantalla, si bien no se trata de un objeto particularmente complejo ya que como tal permite obtener los valores de la pantalla y no manipolarlos como tal, se trata de un elemento que de aprovecharse adecuadamente permite aportar mucho a una paguina web.

Para obtener con las caracteriticas de la pantalla este objeto cuenta con multiples metodos, cada uno con una función en espesifico, los metodos del objeto screen son:

  • .width

    Este metodo permite obtener el ancho total de la pantalla

    Ejemplo

  • .height

    Este metodo permite obtener el alto total de la pantalla

    Ejemplo

  • .availWidth

    Este metodo permite obtener el ancho disponible en la pantalla

    Ejemplo

  • .availHeight

    Este metodo permite obtener el alto disponible en la pantalla

    Ejemplo

  • .pixelDepth

    Este metodo permite obtener la resulución de color de la pantalla

    Ejemplo

  • .colorDepth

    Este metodo permite obtener la profundidad de bits de la pantalla

    Ejemplo
