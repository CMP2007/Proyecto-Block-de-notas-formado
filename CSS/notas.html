Notas

  • Las tres formas de vincular los estilos CSS son:

    1. Como un atributo dentro de la etiqueta en cuestión

    2. Utilizar un alemento "style" para incluirlos dentro de los metadatos de la paguina

    3. Utilizar un elemento "link" para vincular un archivo de estilos css aparte

      4. Nota: la opción recomendable esta resealtada en negritas
